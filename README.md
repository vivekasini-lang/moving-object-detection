# moving-object-detection
Detects moving object in a camera using python and opencv (using contour detection)
## Project description
This project can be used to detect moving object in camera using contour detection and background subtraction and showing text as normal or moving object detected.
## Features
*Real time detection of moving objects
*Texts:Normal if no moving object detected
       Moving object detected if object moves
*Bounding boxes around moving objects 
##Technologies used
*Python
*Opencv
*imutils
## Run the program
1.Install imutils,opencv
2.Run program project 1(moving object detection).py
